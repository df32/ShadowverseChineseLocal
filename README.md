======

#### 主要目的：
- 让翻译流利，通顺；
- 让句子的语法正确；
- 让标点符号使用正确。

#### 主要方向：
- 修正各种病句；
- 修正日文式断句、分句；

- 仅限于『简体中文』的翻译文本。
