翻译通顺化工作
======

#### 主要目的：
- 让语言流利、通顺；
- 让句子的语法正确；
- 让标点符号使用正确。

#### 主要方向：
- 修正各种病句；
- 修正日文式断句、分句；
- 修正日文式标点符号的使用；
- 使句子通顺，易懂。

#### 限定范围：
- 仅限于『简体中文』。

#### 注意：
- 对文本的修改将包括换行的修改，目前还不确定这是否会引起应用错误。
- 所有文本应当维持原有编码格式：包括CodePage、with/without BOM、CR/CRLF。
